**Bruno 3.3.0+** adds first-class **Git providers** so you can search hosts, import collections, and sync without always copying raw Git URLs. **GitHub** is available today; more providers (for example **GitLab**) are planned.

- **[GitHub](/git-providers/github)** — Public import tab, private repos, PAT vs CLI, Enterprise API URL.

For generic Git workflows (init, diff, pull, push) on any remote, see **[Git integration](/git-integration/overview)**.
