# Download Bruno's Desktop Application 

To get started, download and install the latest version of Bruno. 

Visit the [Downloads page](https://www.usebruno.com/downloads) and select the option for your operating system. 

Bruno is available as a native desktop app for macOS, Windows, and Linux.

<Tabs items={['MacOS', 'Windows', 'Linux']}>
  <Tabs.Tab>
    Download Bruno on MacOS from [here](https://www.usebruno.com/downloads).

    To install via Homebrew, run the following command:
    
    ```bash
    brew install bruno
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    Download Bruno on Windows from [here](https://www.usebruno.com/downloads).
    
    For Chocolatey, use the following command:
    ```bash
    choco install bruno
    ```

    For Winget, use the following command:
     ```bash
    winget install bruno
    ```

    For Scoop, use the following command:
     ```bash
    scoop install bruno
    ```

  </Tabs.Tab>
  <Tabs.Tab>
    Download Bruno on Linux from [here](https://www.usebruno.com/downloads).
    
    For Debian-based distributions (like Ubuntu), use the following command:
    
    ```bash
    sudo apt install bruno
    ```

    For Flatpak, use the following command:
    
    ```bash
    flatpak install flathub com.usebruno.Bruno
    ```

    For Snap, use the following command:
    
    ```bash
    sudo snap install bruno
    ```
  </Tabs.Tab>
</Tabs>


The application also comes packaged for these popular distributions:

- **Linux**: snap, apt
- **macOS**: Homebrew
- **Windows**: Chocolatey or Scoop
