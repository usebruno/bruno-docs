Azure Key Vault is a secure and scalable service designed to store and retrieve sensitive information such as API keys, database credentials, and other secrets. 

Bruno allows you to easily integrate with Azure vault and securely access secrets during test execution without exposing them in test scripts or environment variables. 

This guide provides detailed steps to configure and utilize Azure Key Vault in Bruno.

- [Adding a secret provider](/secrets-management/aws-secret-manager/adding-a-secret-provider)
