# Syntax Highlighting Support

Bruno has an editor extension released for VS Code.  You can download it from the [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=bruno-api-client.bruno).

There is also a basic syntax plugin for vim and neovim available at [nikvdp/bruno-vim-syntax](https://github.com/nikvdp/bruno-vim-syntax).

Extensions for IntelliJ and the JetBrains family of IDEs are under development.
