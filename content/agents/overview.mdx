# Overview

Bruno local first approach allows you to configure it with any AI agent and Editors like Cursor, Codex, VS Code, etc. Meanwhile keeping all your data version controlled and stored in your local filesystem. You can configure Bruno to use any AI agent and Editor of your choice.

## Supported Agents

- [Cursor](https://cursor.com)
- [VS Code](https://code.visualstudio.com)
